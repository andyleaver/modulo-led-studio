# HUB75 Feature Pack (ESP32)

This repo version adds:

- ESP32 HUB75 export targets (I2S-DMA)
- Matrix mapping presets (rotate/flip/serpentine)
- Panel order presets + auto-chain
- Clock signals in preview + Rules V6 integration
- Demo projects to validate hardware quickly
- Optional WiFi features in exported sketches:
  - Web updater (OTA)
  - `/info` JSON endpoint
  - mDNS
  - AP fallback WiFi portal
  - NTP time sync

See also: `docs/HUB75_WIFI_OTA.md`
